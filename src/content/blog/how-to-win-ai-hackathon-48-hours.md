## How to Win an AI Hackathon in 48 Hours

Join **Andriy Kostenko**, a Full-Stack Software Engineer, for a practical session on how to design, build, and present a winning AI project in just **48 hours**.

Using **Python + JavaScript + AI** — a battle-tested playbook for speed, impact, and demo-day domination.

Andriy shares the exact strategy he used to win an AI hackathon — from choosing the right idea and tech stack to building a fast MVP and delivering a demo judges remember. Rather than focusing on perfect architecture, this session reveals how to **prioritize speed, clarity, and impact**, using modern AI tools and a streamlined stack.

### What You'll Learn

- **The Winning Mindset:** Why most developers fail at hackathons and how to approach them strategically
- **The Tech Stack That Wins:** Using **FastAPI, React/Next.js, and modern AI models** to build fast prototypes
- **The 48-Hour Execution Plan:** A realistic timeline for turning an idea into a working demo
- **Building a Strong MVP:** Why one impressive feature beats a large unfinished product
- **Demo Strategy:** How to design a presentation that impresses judges and stands out

### Key Topics

- Fast MVP architecture for hackathons
- Integrating AI models into real applications

- Building APIs quickly with FastAPI
- Rapid UI development with React
- Designing demos that create a "wow" moment
